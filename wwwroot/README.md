# Naive-ui

## 安装
### 使用 npm 安装
```cmd
npm i -D naive-ui
```

### 安装 @vicons/ionicons5
@vicons/ionicons5 是一个图标集合，它包含了 Ionicons v5 的图标，并且是以 Vue 组件的形式提供，这样你可以在 Vue 应用中直接使用这些图标。
通过 npm来安装：
```cmd
npm install @vicons/ionicons5
```
